# Vaibhav Yadav - Personal Portfolio

Welcome to the source code for my personal portfolio website! This portfolio highlights my journey, skills, and experience as a Full-Stack Developer and Bug Bounty Hunter based in Greater Noida, India. It's designed to be an "atrangi" (quirky/colorful) representation of my professional persona.

## 🚀 Features

- **Interactive UI/UX**: Custom animations, dynamic neo-brutalism design aesthetics.
- **Experience Timeline**: A visual journey of my professional roles.
- **Skills Matrix**: Detailed breakdown of my technical proficiencies across Frontend, Backend, Architecture, and Security.
- **Terminal Simulator**: An embedded interactive terminal experience.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Design Elements**: Custom SVG graphics, CSS animations, and curated modern typography.

## 💻 Getting Started

To run this project locally:

1. Clone the repository.
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory.
   ```bash
   cd atrangi-portfolio
   ```
3. Provide a local web server to serve the directory. For example, using `serve` via `npx`:
   ```bash
   npx serve .
   ```
4. Open your browser and navigate to `http://localhost:3000` (or whichever port your server is running on).

## 📂 Project Structure

- `index.html`: The main entry point.
- `styles.css` & `neo-styles.css`: The core styling rules and aesthetics for the portfolio.
- `main-script-vy.js`: The core logic for interactions, typing effects, and animations.
- `terminal.html`: An interactive terminal simulator page.
- `image/`: Directory for static assets and images used across the site.
